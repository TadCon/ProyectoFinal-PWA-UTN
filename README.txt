Autor: TADEO CONVERSO

Tema:
Es una aplicación complementaria al juego de mesa "Monopoly", creada para suplir 
el uso de los billetes y llevar registro de las ganancias y las pérdidas a lo largo del juego. 
Cuenta con 3 secciones navegables: 1. Landing page, 2. Creación de jugadores, 
3. Sección del juego

Tecnologías:
Opté por el uso de React para el front por gusto propio y por 
sugerencias de otros en relación a la salida laboral. 
Para el back usé MongoDB sin preferencias en particular, simplemente para continuar 
conociendo otras DBs y no repetir MySQL.
